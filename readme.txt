游戏名称：

步步为营/墙棋/Quoridor


注意事项：
1.config.txt要和quoridor.exe在同一个文件夹下。
2.若发现软件界面超出屏幕，请更改config.txt中的数据，以像素为单位，请不要随意添加空格。默认为700*750。 
3.若按键无效，请检查是否开启了大写锁定，若开启，请关闭。
4.悔棋功能只能在移动棋子或摆放隔板之后下一回合玩家开始行动之前使用。
5.联网对战不能使用悔棋功能。

游戏规则：

玩家轮流行动，选择移动自己的棋子至邻边四格之一，或者摆放一块隔板，隔板将阻挡棋子的移动，但不允许将棋子封闭起来一块板只能占两格。若有其他玩家的棋子相邻，则可跳过后者至后方格，但不能一次跳过两子以上的棋子。若跳过的棋子后方为木板，则可以跳至后者左方或右方。
一方棋子到达对面底线为胜利。


游戏操作：

1.请注意棋盘上方显示的回合信息。
2.该回合玩家若要移动棋子，则可单机自己的棋子，之后棋子闪烁，按wasd控制移动方向，程序自动判断移动是否合法以及是否跳跃。若跳过的棋子后方为木板，则在能斜跳的地方出现棋子，玩家若想斜向跳跃，则用按键选择斜跳方向，若想撤销斜跃，则按q。
3.该回合玩家若要放置隔板且有剩余隔板，则可单机棋盘外己方棕色的隔板标示，则在棋盘左上角出现隔板，用wasd控制隔板移动，隔板自动调整到合法位置，按r旋转隔板，按空格放下隔板。若放下隔板后有棋子被封闭，则弹出对话框提醒重新放置。
4.玩家可在任何时候按棋子或隔板标示进行切换，也可按重新开始。
5.联网对战时按y键输入聊天信息，内容显示在控制台窗口上。
